# API Automation with Playwright

This project demonstrates API test automation using:

- Playwright
- TypeScript
- Postman
- GitHub Actions

## Project Goal

Build a professional API automation framework for portfolio use by following industry-standard practices for:

- test design
- maintainability
- readability
- scalability
- CI integration

## Planned Coverage

- positive test scenarios
- negative test scenarios
- response validation
- basic contract checks
- authentication-related scenarios
- smoke and regression structure
